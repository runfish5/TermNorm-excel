2. Description of Software

TermNorm is an AI-powered terminology normalization tool that:

  • Processes text entered in Microsoft Excel cells
  • Matches free-form text to standardized database identifiers
  • Uses a local backend server for processing
  • May utilize third-party AI/LLM services (such as Groq or OpenAI) and web search APIs
