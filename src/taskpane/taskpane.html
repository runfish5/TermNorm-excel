Change takes effect immediately (session only)

Permanent URL Change:

  1. Go to the TermNorm folder (where you run start-server-py-LLMs.bat)
  2. Open the config folder
  3. Open app.config.json in Notepad
  4. At the very top, add: "backend_url": "http://your-server:8000",
  5. Save the file
  6. Drag it into TermNorm again

Examples:

This computer only: "backend_url": "http://127.0.0.1:8000",
Network computer: "backend_url": "http://192.168.1.100:8000",
