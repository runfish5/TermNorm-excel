AI-Powered Terminology Normalization

Match free-form text to standardized database identifiers using intelligent three-tier matching: exact lookup, fuzzy matching, and LLM research.

Settings

Terminology Legend

Match Methods

Cached - Exact match found in memory cache (instantaneous). Previously matched aliases are stored for fast retrieval.

Fuzzy - Approximate string matching using similarity algorithms (local, offline). No backend required.

ProfileRank - LLM-powered matching with web research. Creates entity profiles and ranks candidates by semantic similarity.

Confidence Scores

0-49 - Low confidence - manual review recommended

50-79 - Medium confidence - likely correct

80-100 - High confidence - very likely correct

Terms

Alias - The input value that needs to be normalized (e.g., "IBM", "Intl Business Machines")

Target ID - The canonical identifier from your reference list (e.g., "International Business Machines Corporation")
